# English

## About

This repository is a personal `chezmoi` dotfiles collection. It stores shell configuration, editor settings, and application configuration files for a Linux environment.

## Contents

- `dot_bash_aliases` — Bash aliases.
- `dot_bashrc` — Bash shell initialization.
- `dot_gitconfig` — Git configuration.
- `dot_profile` — Login profile configuration.
- `dot_config/` — User application settings, including `nvim`, `fish`, `starship`, and custom scripts.
- `private_dot_local/` — Private configuration files and application shortcuts.

## Usage

1. Install `chezmoi` if it is not installed.
2. Use `chezmoi init` and `chezmoi apply` to deploy these dotfiles to your home directory.
3. Review the contents before applying any changes.

## Notes

- This repository is intended for personal configuration management.
- The files are organized to keep shell and editor configuration separate from private or sensitive settings.

# فارسی

<div dir="rtl" align="right">

## درباره

این مخزن مجموعه‌ی شخصی فایل‌های پیکربندی `chezmoi` است. شامل تنظیمات پوسته، ویرایشگر و برنامه‌ها در محیط لینوکس می‌شود.

## محتوا

- `dot_bash_aliases` — میانبرهای Bash.
- `dot_bashrc` — تنظیمات راه‌اندازی Bash.
- `dot_gitconfig` — پیکربندی Git.
- `dot_profile` — پیکربندی پروفایل ورود.
- `dot_config/` — تنظیمات برنامه‌های کاربر، شامل `nvim`، `fish`، `starship` و اسکریپت‌های سفارشی.
- `private_dot_local/` — فایل‌های پیکربندی خصوصی و میانبرهای برنامه.

## نحوه‌ی استفاده

1. اگر `chezmoi` نصب نیست، آن را نصب کنید.
2. از `chezmoi init` و `chezmoi apply` برای استقرار این فایل‌ها در پوشه‌ی خانگی استفاده کنید.
3. قبل از اعمال تغییرات، محتوا را بررسی کنید.

## یادداشت‌ها

- این مخزن برای مدیریت پیکربندی شخصی در نظر گرفته شده است.
- فایل‌ها طوری سازمان‌دهی شده‌اند که تنظیمات پوسته و ویرایشگر از تنظیمات خصوصی یا حساس جدا بمانند.
